Commands

View all commands in devilbot across moderation and Devil Cash.

All Categories
Prefix: .

Moderation

/ban

Ban a member with optional history delete.

Arguments: <user> [reason]
/kick

Kick a member with reason.

Arguments: <member> [reason]
/softban

Ban then unban to clear messages.

Arguments: <member> [history] [reason]
/massban

Ban multiple IDs/mentions at once.

Arguments: <users> [reason]
/unbanall

Unban every banned user.

Arguments: none
/mute

Timeout a member.

Arguments: <member> [minutes]
/unmute

Remove timeout from a member.

Arguments: <member>
/reactionmute

Block reactions in a channel.

Arguments: <member> <channel>
/unreactionmute

Allow reactions again.

Arguments: <member> <channel>
/lock / /unlock

Lock/unlock the current channel.

Arguments: [reason]
/unlockdown / /unlockdownall

Unlock specific or all text channels.

Arguments: <channel> [reason] / none
/bc

Clear bot messages in the channel.

Arguments: none
/purge

Delete up to 500 recent messages.

Arguments: <count>
/history

Show warns and Devil Cash for a user.

Arguments: <user>
/forcenickname

Set or clear a member’s nickname.

Arguments: <member> [nickname]
/role / /r

Add/remove/toggle roles quickly.

Arguments: <user> <role> [add|remove|toggle]
/picperms

Toggle attachments for a user in a channel.

Arguments: <channel> <user>
/newusers

List newest members.

Arguments: [amount]
/setnotifications

Set server default notifications.

Arguments: <option>
/unlock

Unlock the current channel.

Arguments: none
/unlockdown

Unlock a specific channel.

Arguments: <channel> [reason]
/unlockdownall

Unlock all text/announcement channels.

Arguments: [reason]
/bc

Clear bot messages in the channel.

Arguments: none
/role

Add or remove a role from a user.

Arguments: <user> <role> [add|remove]
/r

Alias for /role (same arguments).

Arguments: <user> <role> [add|remove]

Antinuke

/antinuke enable/disable

Toggle anti-nuke protection.

Arguments: none
/antinuke status

Show anti-nuke status, whitelist, and limits.

Arguments: none
/antinuke whitelist

Manage whitelist (add/remove/list).

Arguments: action, [user]
/antinuke limits

Set ban/kick/channel delete thresholds.

Arguments: [ban] [kick] [channel]

Devil Cash

/devildaily

Claim daily Devil Cash (24h cooldown).

Arguments: none
/devilhunt

Harvest DC with a chance of bonus.

Arguments: none
/devilpray

Gain or lose DC at random.

Arguments: none
/devilcurse

Steal DC from another user.

Arguments: <target>
/devilbalance

Check your DC balance.

Arguments: none
/devilgamble

Coinflip (heads/tails) for DC.

Arguments: <amount> <heads|tails>
/devilblackjack

Interactive blackjack with hit/stand.

Arguments: <bet>
/devilbj

Alias for /devilblackjack.

Arguments: <bet>
/help

List bot help and support info.

Arguments: none
